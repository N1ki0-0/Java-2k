 ---- Drink и Dish ----
Drink представляет напиток и реализует интерфейс MenuPosition.
Этот класс хранит информацию о стоимости, названии, описании, содержании алкоголя и типе напитка.
Dish представляет блюдо и также реализует интерфейс MenuPosition.
Этот класс хранит информацию о стоимости, названии и описании блюда.
 ---- MenuPosition ----
Это интерфейс, который объявляет методы для получения информации о позиции меню,
такие как стоимость, название и описание. Классы Drink иDish
 ---- Order ----
Этот класс представляет заказ и имеет методы для управления позициями в заказе, подсчета стоимости и количества позиций.
 ---- TableOrder и InternetOrder ----
TableOrder представляет заказ, сделанный в ресторане и хранит список позиций заказа, реализует Order
InternetOrder представляет заказ, сделанный через интернет, и также хранит список позиций заказ, реализует Order
 ---- Customer ----
представляет информацию о клиенте, такую как имя, возраст и адрес
 ---- Address ----
представляет адрес клиента
 ---- DrinkTypeEnum ----
список напитков
